🎬 Movie Genre Classification

This project is a Machine Learning model developed as part of my internship Task-1. The model predicts the genre of a movie based on its plot summary using Natural Language Processing (NLP) techniques and text classification algorithms.

📌 Project Description

The Movie Genre Classification system is a supervised machine learning model that analyzes textual movie descriptions and automatically classifies them into appropriate genres.

The model uses NLP techniques to convert text into numerical features and applies classification algorithms to make accurate predictions.

🔎 It solves the problem of manual genre tagging by automating the classification process efficiently.

🚀 Features

✔️ Text Preprocessing (cleaning, normalization, stopword removal)
✔️ TF-IDF Feature Extraction
✔️ Multiple ML Models (Naive Bayes, Logistic Regression, SVM)
✔️ Model Evaluation with Accuracy, Confusion Matrix & Classification Report
✔️ Comparison of model performance
